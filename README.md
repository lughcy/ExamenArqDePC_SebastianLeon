# ExamenArqDePC_SebastianLeon

## Objetivo

Desplegar una infraestructura en AWS utilizando CloudFormation y GitHub Actions.

## Tecnologías Utilizadas

- AWS EC2
- AWS CloudFormation
- GitHub Actions
- Git Flow
- HTML
- CSS
- JavaScript

## Deploy

El workflow Deploy valida y despliega una instancia EC2 mediante CloudFormation.

## Destroy

El workflow Destroy elimina completamente el stack desplegado.

## Template EC2

El template crea:

- 1 Instancia EC2 Amazon Linux
- 1 Security Group
- Puerto 80 habilitado
- Página web personalizada con HTML, CSS y JavaScript
